CRITICAL NOTICE: Snapshots are NOT RAW DATA. They contain computed values from API responses and have NO SCIENTIFIC VALUE ALONE for raw data analysis. Use RAW DATA for scientific reproducibility.

What are Snapshots?

SNAPSHOT data includes:

  • API responses from official agencies (e.g., JPL SBDB API)
  • Computed orbital elements
  • Real-time queries
  • Summary statistics

Snapshots are NOT:

  • Original archival files
  • Direct downloads from mission archives
  • Raw observational data

Scientific Usage

For scientific reproducibility and peer review, researchers should use RAW DATA from official mission archives, not SNAPSHOT files.

Snapshots may be useful for:

  • Monitoring changes in computed orbital solutions over time
  • Quick reference for current orbital parameters
  • Cross-referencing with RAW observational data

However, they must NOT be used as primary evidence in scientific publications without citing the original RAW data sources.

Communication Policy: TRIZEL-AI does not provide direct messaging. All communication is conducted through verifiable channels: GitHub (issues, pull requests, discussions), ORCID (research profiles), and DOI-linked archival records (Zenodo, OSF, HAL).
