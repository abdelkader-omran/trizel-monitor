Overview

TRIZEL Monitor is an automated system designed to track, collect, and archive data related to the interstellar object 3I/ATLAS from authoritative space agencies.

The system operates with zero daily manual intervention, relying on GitHub Actions to execute scheduled and on-demand monitoring workflows.

Core Principles
  • No interpretation: System does not modify, reinterpret, or alter official datasets
  • Evidence-first: Pointers + verifiable provenance only
  • Strict classification: Clear separation of RAW_DATA vs SNAPSHOT vs DERIVED
  • Reproducible: All automation logic is auditable and reproducible
